CS Dev Final Project Website

TL;DR—The Basic Requirements

Instructors will create a hand-coded, dynamic website with

  1. HTML
  2. CSS
  3. JavaScript

The Rationale

We want all instructors to have

  • knowledge and experience to bring into the classroom
  • skills and projects to take with them after 9 Dots

Building a website gives instructors a parallel learning path to their students: both are learning new and complex computational thinking. And the real-world nature of web development means the skills instructors learn (and the evidence that they learned it) will be useful beyond 9 Dots.

A Note on Computational Thinking

Working on a significant project will exercise computational thinking. HTML and CSS alone will require organization and attention to detail, which tie into computational thinking elements like decomposition, pattern recognition, and abstraction. Adding JavaScript to the mix will take those to another level, and add algorithms to the mix.

FAQ

Really, those are the only requirements? I won’t be dinged for doing only the basics?

This project intentionally has simple base requirements. Our hope is that this high flexibility leads to high personal investment—which allows you to find an area where you can challenge yourself, and then rise to that challenge. And the more you push yourself, the more we can use that experience to help you figure out where you might grow next in your CS development journey.

What do you mean by dynamic website?

We mean that via user interaction, the contents of the website itself change. Using something like a :hover CSS pseudo-class to temporarily change the appearance of something on a website does not count, because it only changes the presentation. Using JavaScript to change CSS classes on a website element in response to a button press does count, as does using JavaScript to insert/delete/modify website content. Please note: a dynamic website requires JavaScript.*

*Unless you want to do something hardcore.

Must I write all my website code from scratch?

We expect that you will use and modify code from other sources. We just ask that you attribute those sources. Your portions of the code, though, should be significant enough that you can call the entire project yours.
